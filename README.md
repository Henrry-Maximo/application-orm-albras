# SGA - Sistema de Gestão by Albras (Prototype UI)

Este repositório contém o protótipo da interface de autenticação para o **SGA (Sistema de Gestão by Albras)**. 
O objetivo deste projeto foi demonstrar uma proposta de modernização visual e melhoria na experiência do usuário (UX) em comparação à interface anterior.

---

## 🚀 Proposta do Projeto

O foco principal foi criar uma interface **Clean**, responsiva e intuitiva, seguindo padrões modernos de design para sistemas corporativos. 

- **Foco em UI/UX:** Tipografia legível, contraste balanceado e paleta de cores sóbria.
- **Performance:** Construído com React para garantir transições fluidas.
- **Iniciativa:** Projeto desenvolvido como prova de conceito para melhoria de sistemas internos.

## 🛠️ Tecnologias Utilizadas

- **React.js**: Biblioteca principal para construção da interface.
- **CSS3 / Styled Components**: Estilização modular e responsiva.
- **Vite/React Scripts**: Tooling para desenvolvimento rápido.
- **Vercel**: Deploy automatizado e hospedagem.
